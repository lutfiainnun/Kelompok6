**BAB I Pendahuluan**
----------
1.1 Tujuan
----------
Dokumen Software Requirement Specification (SRS) merupakan dokumen spesifikasi perangkat lunak untuk membangun "Sistem Pengeloaan Edotel SMKN 1 Pekanbaru". Dokumen ini dibangun untuk memudahkan pihak sekolah untuk menginput data-data mengenai stock persediaan barang yang ada pada unit produksi laundry SMKN 1 Pekanbaru. Sehingga dokumen ini dapat dijadikan acuan teknis untuk membangun perangkat lunak "Inventory Laundry SMKN 1 Pekanbaru".

1.2   Lingkup
----------
Sistem Inventory Laundry merupakan website yang kami bangun untuk mempermudah staff dalam mengelola persediaan barang maupun barang yang dibutuhkan dalam menjalankan usaha laundry yang ada di SMKN 1 Pekanbaru seperti deterjen, pelembut kain, bahan kimia pencucian, pengharum pakaian, hanger, hingga kemasan.
